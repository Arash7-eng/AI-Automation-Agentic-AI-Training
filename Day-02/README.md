# Practice Tasks

## Task 1: Zero-Shot Prompting

### Objective

Create prompts without giving examples.

### Example

Prompt:
Translate the following English sentence into Hindi:
"I love programming."

Output:
मुझे प्रोग्रामिंग पसंद है।

---

## Task 2: One-Shot Prompting

### Objective

Provide one example and ask AI to follow the pattern.

### Example

Input: Cat
Output: Animal

Input: Rose
Output: Flower

---

## Task 3: Few-Shot Prompting

### Objective

Provide multiple examples for better accuracy.

### Example

Input: Dog → Mammal
Input: Eagle → Bird
Input: Snake → Reptile

Input: Frog
Output: Amphibian

---

## Task 4: Sentiment Analysis

### Objective

Classify reviews as Positive or Negative.

### Example

Review:
"The phone is amazing and battery lasts long."

Output:
Positive

Review:
"The laptop hangs frequently and customer support is poor."

Output:
Negative

---

## Task 5: English to SQL

### Objective

Convert natural language into SQL queries.

### Example

Question:
Show all students whose marks are greater than 80.

SQL:
SELECT * FROM students
WHERE marks > 80;

---

## Task 6: Information Extraction

### Objective

Convert text into JSON format.

### Example

Input:
Name: Arash
Age: 19
Course: AI

Output:
{
"name": "Arash",
"age": 19,
"course": "AI"
}

---

## Task 7: Chain of Thought Prompting

### Objective

Solve problems step by step.

### Example

Prompt:
A shopkeeper buys a pen for ₹20 and sells it for ₹25.
Calculate profit percentage.
Think step by step.

---

## Task 8: Role Prompting

### Objective

Assign a specific role to AI.

### Example

Prompt:
You are a Python Teacher.
Explain Lists, Tuples, and Dictionaries using simple language and examples.

---
