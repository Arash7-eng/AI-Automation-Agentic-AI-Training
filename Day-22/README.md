# 🚀 Project 1 – AI Career Counseling Multi-Agent System

## 📖 Description

The **AI Career Counseling Multi-Agent System** is an intelligent assistant that helps students identify suitable career paths based on their education, technical skills, interests, and future goals.

The system accepts user input at runtime and passes the information through multiple AI agents. Each agent performs a specific task, and together they generate a personalized career guidance report.

---

## 🎯 Objectives

- Analyze student profiles
- Evaluate current technical skills
- Recommend career opportunities
- Generate personalized learning roadmaps
- Suggest online learning resources
- Track career readiness
- Produce a final AI-generated career report

---

# 🚀 Project 2 – AI Multi-Agent Project Planning System

## 📖 Description

The **AI Multi-Agent Project Planning System** converts a software project idea into a complete development plan using multiple collaborating AI agents.

Instead of manually preparing project documentation, different AI agents work together to generate a professional project plan including requirements, architecture, timeline, modules, risks, and reviewer feedback.

---

## 🎯 Objectives

- Recommend software architecture
- Generate technology stack
- Create development timeline
