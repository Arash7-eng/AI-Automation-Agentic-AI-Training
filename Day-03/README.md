# Day 03 - Modern Web Application Development Projects

## Overview

As part of my Day 03 training, I designed and developed three modern web applications focused on solving real-world student and personal productivity challenges. These projects demonstrate my understanding of frontend web development, responsive user interface design, local data management, and multi-page application architecture using Lovable and Replit.

The projects were built with a strong emphasis on usability, clean design, and practical functionality while applying core web development concepts such as CRUD operations, local storage, responsive layouts, dashboard creation, and user-centered design.

---

## Project 1: Pennywise - Daily Expense Tracker

Pennywise is a modern personal finance management application that enables users to efficiently track income, expenses, and monthly budgets through an intuitive dashboard.

### Key Features

* Multi-page application architecture
* Dashboard with financial overview
* Income and expense management
* Transaction history and filtering
* Budget tracking with progress indicators
* Category-wise expense analysis
* User profile management
* Currency selection support (₹, $, £)
* Dark and Light Mode
* Responsive design for desktop and mobile devices

### Skills Applied

* Dashboard Design
* Financial Data Visualization
* Responsive UI Development
* User Experience Design
* Theme Management

## Project 3: College Attendance Tracker

A student-focused attendance management system designed to monitor subject-wise attendance and academic progress.
