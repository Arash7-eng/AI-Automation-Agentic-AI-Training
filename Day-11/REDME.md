# Day 11 – Evolution API Setup & Account Configuration

## Objective
The objective of this task was to set up the Evolution API environment and connect it with WhatsApp. This is the first step before building the AI-powered WhatsApp Personal Assistant in n8n.

## Steps Performed

### 1. Created a Cloud Station Account
- Opened the Cloud Station website.
- Registered a new account using an email address.
- Logged into the Cloud Station dashboard.

### 2. Created a Project
- Created a new project (or used the automatically generated project).
- Opened the project dashboard.

### 3. Deployed Evolution API
- Navigated to **App Templates**.
- Searched for **Evolution API**.
- Clicked **Deploy Now** to start the deployment.
- Waited for the deployment process to complete.

### 4. Connected WhatsApp
- Opened the deployed Evolution API service.
- Generated the QR Code.
- Opened WhatsApp on the mobile device.
- Navigated to **Linked Devices**.
- Scanned the QR Code to connect the WhatsApp account.

### 5. Opened Evolution API Manager
- Opened the Evolution API service URL.
- Accessed the **Manager** page.
- Retrieved the authentication password from the **Variables** section.
- Logged into the Evolution API Manager successfully.

## Outcome
- Successfully created a Cloud Station account.
- Successfully deployed the Evolution API.
- Connected WhatsApp with Evolution API.
- Logged into the Evolution API Manager.
- Environment is now ready for integrating with n8n in the next step.

## Technologies Used
- Cloud Station
- Evolution API
- WhatsApp
