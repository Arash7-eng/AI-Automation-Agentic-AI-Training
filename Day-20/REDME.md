# 🚀 Day 20 – CampusBot: AI Student Helpdesk Agent

## 📖 Project Overview

On **Day 20** of my **AI Automation & Agentic AI**, I developed **CampusBot**, an AI-powered Student Helpdesk Agent designed to simplify communication between students and the college administration. The project demonstrates how prompt engineering and conversational AI can be used to create a virtual assistant capable of answering common college-related questions in a professional, accurate, and user-friendly manner.

CampusBot acts as a digital assistant for **ABC College**, helping students, prospective students, and parents by providing information about admissions, examinations, fee payments, library services, timetables, campus events, department locations, and frequently asked questions.

Unlike a general AI assistant, CampusBot follows a strict role-based prompt that ensures it only answers college-related queries. If a user asks something outside its scope, the chatbot politely explains its limitations and encourages the user to ask a college-related question instead.

# 🧠 Prompt Engineering

A detailed system prompt was created to define the behavior of CampusBot. The prompt ensures that the chatbot remains focused on college-related topics while maintaining accuracy and professionalism.

CampusBot follows these rules:

- Answers only ABC College-related questions.
- Never generates false or misleading information.
- Politely declines unrelated questions.
- Protects student privacy.
- Asks clarification questions whenever required.
- Suggests the correct college department if additional support is needed.

This role-based prompt makes the chatbot more reliable and suitable for educational environments.
